# SORTITO Walkthrough – Step by Step

This guide walks you through using SORTITO, from launching the app to finishing your first media sort.

## 📥 Getting SORTITO

### Option A: Pre‑built executable (Windows)
1. Download `SORTITO.exe` from the [Releases](../../releases) page.
2. Save it anywhere (e.g., Desktop, Downloads folder).
3. Double‑click to run – no installation needed.

### Option B: Run from Python source
```bash
git clone https://github.com/yourusername/sortito.git
cd sortito
pip install customtkinter pillow
python SORTITO.py
